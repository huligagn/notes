# DTO 转换逻辑对比

|    转换方式     |             优点             |          缺点          |                     适用场景                      |
| :-------------: | :--------------------------: | :--------------------: | :-----------------------------------------------: |
| Getter & Setter | 性能好，可定制每个字段的转换 | 代码逻辑较多，容易遗漏 | 字段少于5第一选择，担心性能有问题可以选择这个方案 |
|                 |                              |                        |                                                   |
