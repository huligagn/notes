# Building Spring web applications

对于java程序员来说，主要工作大部分是web开发，状态控制、工作流、验证等等都是需要重点关注的特性，然而这些特性要基于无状态的http协议来实现就不是那么简单了。

spring的网络框架是为了解决这些任务专门设计的，Spring MVC可以帮助建立灵活、低耦合的web程序。

这一章主要讲解Spring MVC框架。
